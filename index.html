Control MOD por etapa

Formato editable generado desde Formato para MOD.xlsx. Cambia semanas, STD, HC balance u HC óptimo y la gráfica se actualiza automáticamente. El tema visual fue ajustado con la referencia de colores rojos/anaranjados y la imagen de TATA.

Supabase listo
Archivo compartible / link opcional
El link opcional copia el estado actual en la URL. Si el link resulta demasiado grande, exporta CSV para respaldo de datos.
La tabla STD es el estándar de personas por área. Puedes copiar una fila horizontal desde Excel y pegarla en la primera celda; se llenará toda la fila automáticamente. Los cambios se conservan automáticamente en este navegador. La línea naranja de cada gráfica toma el total STD de la etapa; las barras muestran HC Balance y HC Óptimo.
